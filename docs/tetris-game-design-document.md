# 俄罗斯方块游戏设计文档

## 1. 项目概述

### 1.1 项目名称
Tetris-Game2 - 基于 HTML5 Canvas 的网页版俄罗斯方块游戏

### 1.2 项目目标
开发一款经典的俄罗斯方块游戏，具备完整的游戏机制、流畅的操作体验和现代化的界面设计。

### 1.3 技术栈
- **前端**: HTML5, CSS3, JavaScript
- **渲染**: Canvas 2D API
- **存储**: LocalStorage
- **音效**: Web Audio API

## 2. 游戏核心机制

### 2.1 游戏规则
1. **方块下落**: 七种不同形状的方块从顶部下落
2. **方块控制**: 玩家可以移动、旋转方块
3. **消除机制**: 水平填满一行时消除该行
4. **游戏结束**: 方块堆叠到顶部时游戏结束
5. **得分系统**: 根据消除行数和游戏难度计算得分

### 2.2 方块类型 (Tetromino)
| 方块名称 | 形状 | 颜色 | 旋转状态数 |
|---------|------|------|-----------|
| I-块 | 直线型 | 青色 | 2 |
| O-块 | 正方形 | 黄色 | 1 |
| T-块 | T字型 | 紫色 | 4 |
| S-块 | S字型 | 绿色 | 2 |
| Z-块 | Z字型 | 红色 | 2 |
| J-块 | J字型 | 蓝色 | 4 |
| L-块 | L字型 | 橙色 | 4 |

### 2.3 操作控制
- **左右移动**: ← → 方向键
- **加速下落**: ↓ 方向键
- **旋转**: ↑ 方向键 或 空格键
- **暂停/继续**: P 键或 ESC 键
- **重新开始**: R 键

### 5.2 得分系统
- **单行消除**: 100 × 等级
- **双行消除**: 300 × 等级
- **三行消除**: 500 × 等级
- **四行消除**: 800 × 等级 (Tetris)
- **软降**: 1分/格
- **硬降**: 2分/格

### 5.3 等级系统
- **等级提升**: 每消除10行提升1级
- **下落速度**: 随等级增加而加快
- **速度公式**: `speed = Math.max(50, 1000 - level * 50)` (毫秒)

## 7. 性能要求

### 7.1 性能指标
- **帧率**: 保持60FPS稳定运行
- **响应时间**: 输入延迟 < 16ms
- **内存使用**: < 50MB
- **加载时间**: < 3秒

### 7.2 兼容性要求
- **浏览器**: Chrome 60+, Firefox 55+, Safari 12+, Edge 79+
- **移动设备**: iOS 12+, Android 7.0+
- **屏幕分辨率**: 支持320px - 1920px宽度

### 7.3 优化策略
1. **Canvas优化**: 使用离屏Canvas缓存静态元素
2. **动画优化**: 使用requestAnimationFrame
3. **内存管理**: 及时清理不用的对象引用
