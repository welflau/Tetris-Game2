# 俄罗斯方块游戏设计文档

## 1. 游戏概述

### 1.1 项目信息
- **游戏名称**: Tetris-Game2
- **游戏类型**: 经典益智消除类游戏
- **开发平台**: 网页端 (HTML5)
- **技术栈**: HTML5 Canvas + JavaScript ES6+
- **目标用户**: 所有年龄段的休闲游戏玩家
- **开发周期**: 4.5周

### 1.2 游戏目标
- 提供经典俄罗斯方块的完整游戏体验
- 支持现代浏览器的流畅运行
- 简洁直观的用户界面
- 响应式设计，支持不同屏幕尺寸

### 1.3 核心特色
- 🎮 经典的俄罗斯方块玩法
- ✨ 流畅的动画效果
- 📊 实时计分系统
- 🏆 本地最高分记录
- 📱 响应式设计

## 2. 核心玩法机制

### 2.1 方块类型
游戏包含7种经典方块类型：

| 方块名称 | 形状 | 颜色 | 旋转状态数 |
|---------|------|------|----------|
| I-型 | 直线型 | 青色 | 2 |
| O-型 | 正方形 | 黄色 | 1 |
| T-型 | T字形 | 紫色 | 4 |
| S-型 | S字形 | 绿色 | 2 |
| Z-型 | Z字形 | 红色 | 2 |
| J-型 | J字形 | 蓝色 | 4 |
| L-型 | L字形 | 橙色 | 4 |

### 2.2 游戏规则

#### 2.2.1 基础操作
- **左右移动**: 方向键 ← →
- **下降加速**: 方向键 ↓
- **旋转**: 方向键 ↑ 或 空格键
- **暂停/继续**: P 键
- **重新开始**: R 键

#### 2.2.2 消除机制
- 当水平行完全填满时，该行消除
- 可同时消除多行（最多4行）
- 消除后上方方块自动下落
- 连续消除有额外奖励分数

#### 2.2.3 游戏结束条件
- 新方块无法在顶部正常生成时游戏结束
- 方块堆积到游戏区域顶部

### 2.3 计分系统

#### 2.3.1 基础分数
- 单行消除: 100分 × 当前等级
- 双行消除: 300分 × 当前等级
- 三行消除: 500分 × 当前等级
- 四行消除: 800分 × 当前等级

#### 2.3.2 等级系统
- 每消除10行升1级
- 等级影响方块下降速度
- 最高等级: 15级
