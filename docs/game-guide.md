# 🎮 俄罗斯方块游戏指引

欢迎来到俄罗斯方块游戏！这是一个经典的益智游戏，通过旋转和移动下落的方块来消除完整的行。

## 🎯 游戏目标

- 通过移动和旋转下落的方块，填满水平行
- 填满的行会自动消除并获得分数
- 尽可能长时间地保持游戏继续，获得更高分数
- 当方块堆积到顶部时游戏结束

## 🕹️ 操作控制

### 键盘控制
| 按键 | 功能 |
|------|------|
| **A** 或 **←** | 向左移动 |
| **D** 或 **→** | 向右移动 |
| **S** 或 **↓** | 快速下落 |
| **W** 或 **↑** | 旋转方块 |
| **空格键** | 暂停/继续游戏 |

### 操作技巧
- 使用**投影方块**（半透明显示）来预判最佳落点
- 合理利用**旋转**功能来适应不同的空间
- **快速下落**可以节省时间，获得额外分数
- 观察**下一个方块预览**来制定策略

## 🧩 方块类型

游戏包含7种经典的俄罗斯方块（Tetromino）：

| 方块 | 形状 | 特点 |
|------|------|------|
| **I** | ████ | 直线型，最适合消除4行 |
| **O** | ██<br>██ | 正方形，旋转不变 |
| **T** | ███<br>&nbsp;█ | T字型，用途广泛 |
| **S** | &nbsp;██<br>██ | S字型，注意旋转方向 |
| **Z** | ██<br>&nbsp;██ | Z字型，S的镜像 |
| **J** | █<br>███ | J字型，适合填补角落 |
| **L** | &nbsp;&nbsp;█<br>███ | L字型，J的镜像 |

## 📊 计分系统

### 基础得分
- **单行消除**：100 分 × 当前等级
- **双行消除**：300 分 × 当前等级  
- **三行消除**：500 分 × 当前等级
- **四行消除（Tetris）**：800 分 × 当前等级

### 额外得分
- **软降落**：每格 1 分（使用 S/↓ 键）
- **硬降落**：每格 2 分（直接落到底部）

### 等级提升
- 每消除10行提升1个等级
- 等级越高，方块下落速度越快
- 高等级获得的分数倍数更高

## 🎮 游戏界面

### 主要区域
1. **游戏区域**：10×20的方格，方块在此下落和堆积
2. **下一个方块**：显示即将出现的方块类型
3. **分数显示**：当前分数、等级、消除行数
4. **投影方块**：半透明显示方块的预期落点

### 状态信息
- **分数（Score）**：当前总分数
- **等级（Level）**：当前游戏等级
- **行数（Lines）**：已消除的总行数
- **下一个（Next）**：下一个方块预览

## 🏆 游戏策略

### 初级策略
1. **保持底部平整**：避免留下深坑
2. **优先消除多行**：等待机会一次消除多行
3. **合理使用T字块**：T字块最灵活，可以填补各种空隙
4. **留意I字块**：I字块是消除4行的关键

### 高级技巧
1. **T-Spin技术**：利用T字块的旋转特性获得额外分数
2. **堆叠策略**：在一侧留出4格宽的空间等待I字块
3. **预判组合**：根据下一个方块预览制定放置策略
4. **速度控制**：在高等级时保持冷静，不要急于下落

## ⚠️ 注意事项

### 游戏结束条件
- 当新方块无法完全进入游戏区域时游戏结束
- 通常发生在方块堆积到顶部时

### 避免常见错误
- 不要在游戏区域中央留下深坑
- 避免只消除单行，尽量等待多行消除机会
- 不要忽视投影方块的提示
- 注意方块的旋转中心点
